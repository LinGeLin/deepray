Welcome to DeePray's documentation!
====================================

**DeePray** is a LEGO for deep learning algorithms. It is designed to make AI easier, faster and cheaper with the following advantages:

- Faster training speed and higher efficiency.
- Lower memory usage.
- Better accuracy.
- Support of parallel, distributed, and GPU learning.
- Capable of handling large-scale data.

For more details, please refer to `Features <./Features.rst>`__.
